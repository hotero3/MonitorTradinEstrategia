# 🚀 Hotero Pro Dashboard - Crypto Trading Tools

Sistema de monitoreo para BTCUSDT que integra indicadores técnicos de Google Sheets y datos en tiempo real de Binance.

### 🛠 Tecnologías
- **Frontend:** HTML5, CSS3 (Flexbox), JavaScript (ES6+).
- **Backend:** Google Apps Script (GAS).
- **Gráficos:** Chart.js.
- **Fuentes de datos:** CoinGlass API, Binance API, Google Sheets.

### 📈 Características
- Monitoreo de **MACD, ADX y DMI**.
- Alerta visual y auditiva de **Cruces de MACD**.
- Seguimiento de **Delta Long/Short** en tiempo real.
- Panel de control de **PnL Personal** para simulación de entradas.
